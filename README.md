## Technologies Used
- Python
- Django
- HTML, CSS, Bootstrap
- SQLite

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run migrations:
   python manage.py migrate
4. Start server:
   python manage.py runserver
